# 📌 프로젝트명: SmartFit — AI 기반 개인 맞춤형 운동 추천 시스템

AI를 활용해 운동 루틴을 자동 추천하고, 실시간 피드백까지 제공하는 웹 애플리케이션 프로젝트입니다.

---

# 🔑 핵심 개념 정리

| **핵심 개념** | **설명** | **추가 정보** | **예시** |
|--------------|----------|----------------|-----------|
| **User Profiling** | 사용자의 신체·운동 데이터 분석 | InBody, 운동 빈도 등 | "주 3회 운동, 체지방률 25%" |
| **Recommendation Engine** | AI가 맞춤 운동 루틴 생성 | 협업필터링+룰 기반 | 상체 루틴 자동 추천 |
| **Workout Feedback** | 자세·운동량 분석 후 피드백 제공 | 센서/로그 기반 | "속도 10% 감소 필요" |
| **Progress Tracking** | 주차별 변화 시각화 | 그래프/리포트 | 체중 변화 그래프 |
| **Gamification** | 운동 지속성 향상 요소 | 뱃지·레벨 시스템 | "연속 7일 달성" |

---

# ✔️ 3단계 체크리스트

### 1) 데이터 수집
- 사용자 프로필 입력  
- 운동 기록 저장  
- 목표 설정(다이어트/근력/체력)

### 2) AI 추천 엔진
- 유사 사용자 기반 추천  
- 목표 기반 난이도 조절  
- 자동 루틴 편성  

### 3) 결과 모니터링
- 주차별 리포트  
- 그래프 분석  
- 성과 기반 피드백 루프  

---

# 🚀 주요 기능

### ✅ 개인 맞춤 운동 추천  
- 신체 정보 기반 난이도 조절  
- 헬린이/중급/상급 자동 분류  

### ✅ 실시간 운동 피드백  
- 속도·자세·횟수 분석  
- 실시간 코칭 메시지 제공  

### ✅ 주차별 성과 리포트  
- 체중/근력/운동시간 추이  
- 개선 포인트 자동 생성  

### ✅ 게임화 요소  
- 뱃지, 레벨 시스템  
- 연속 운동일 기록 등
